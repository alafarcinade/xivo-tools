*** Mise en place de l'IP virtuelle  ***
========================================


1) Sur le maître:
 - copier les scripts ac-ha-manage-virtual-ip et relay-slave dans /usr/local/sbin
 - configurer ac-ha-manage-virtual-ip correctement
 - dans /etc/cron.d/xivo-ha, ajouter la ligne suivante:
    * * * * * root /usr/local/sbin/relay-slave IP_VIRTUELLE_VOIP >/dev/null
 - copier dhcpd_ha_master.conf dans /etc/dhcp
 - dans dhcpd.conf, ajouter les lignes suivantes avant les 2 derniers includes:
        #HA configuration
        include "/etc/dhcp/dhcpd_ha_master.conf";
 - dans dhcpd_subnet.conf et dhcpd_subnet.conf.head, rajouter la ligne suivante après l'instruction "range":
        failover peer "dhcp-failover";

2) Sur l'esclave:
 - copier le script ac-ha-manage-virtual-ip dans /usr/local/sbin
 - configurer ac-ha-manage-virtual-ip correctement
 - remplacer xivo-check-master-status par la version présente dans xivo-tools
 - activer le DHCP, avec les mêmes paramètres que le maître
 - copier dhcpd_ha_slave.conf dans /etc/dhcp
 - dans dhcpd.conf, ajouter les lignes suivantes avant les 2 derniers includes:
	#HA configuration
	include "/etc/dhcp/dhcpd_ha_slave.conf";
 - dans dhcpd_subnet.conf et dhcpd_subnet.conf.head, rajouter la ligne suivante après l'instruction "range":
	failover peer "dhcp-failover";
